## 外设
### lcd
spi1接口
### 按键*6
独立按键
### EC11
旋转编码器 带按键
### 485
485通讯接口
### CAN
CAN通讯接口
### CD4052
板上只有一个接口，通过CD4052来切换485 和 CAN 以及他们的极性
### 红外
红外发射 接收
### SD卡
spi2接口SD卡，并引出SD检测脚
